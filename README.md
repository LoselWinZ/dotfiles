# Install Dependencies

```
sudo dnf install stow
sudo dnf install git
```

# Installation

Checkout this repo to your $HOME directory using git.

```
git clone git@github.com:LoselWinZ/dotfiles.git
cd dotfiles
```

then use GNU Stow to create symlinks

```
stow .
```
